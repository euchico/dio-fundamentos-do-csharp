## Criando uma solução via CLI

Atalho Windows:
- Ctrl+E -> Abrir Windows Explorer.
*(!) Digitar 'cmd' na barra de endereço do Windows Explorer para abrir o prompt na referente pasta.*

Comandos Prompt Windows:
- `cd` -> Entrar na pasta.
- `dir` -> Lista os arquivos do diretório.

Comandos CLI:
- `dotnet new list` -> Lista de modelos de arquivos.
- `dotnet new sln -n {NOME}` -> Cria uma solução.
	*Ex.: dotnet new sln Curso*

---

## Criando um projeto

Comandos CLI:
- `dotnet new {MODELO} -n {NOME} -f {VERSÃO .NET}` -> Cria um Projeto.
	*Ex.: dotnet new console -n ProjetoAulas -f net9.0*

---

## Executando o primeiro projeto

Comandos Prompt Windows:
- `type {ARQUIVO}` -> Descreve o arquivo.
	*Ex.: type Curso.sln*
- `cls` -> Limpa o prompt.

Comandos CLI:
- `dotnet sln {SOLUÇÃO} add {PROJETO}` -> Adiciona um Projeto à Solução.
	*Ex.: dotnet sln Curso.sln add ProjetoAulas
- `dotnet build` -> Compila a solução.
- `dotnet clean` -> Limpa as pastas de arquivos temporários.
- `dotnet run --project {NOME DO PROJETO}` -> Executa o projeto.
	*Ex.: dotnet run --project ProjetoAulas*
